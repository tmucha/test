# Welcome

This repository contains a simple FastAPI service that renders a Hello World message via an HTML template.

## Getting Started

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the development server:
   ```bash
   uvicorn app.main:app --reload
   ```
3. Open your browser and navigate to `http://127.0.0.1:8000/` to view the Hello World page.
